============
Installation
============


Stable release
--------------

To install the Psi4 plug-in, run this command in your terminal:

.. code-block:: console

    $ pip install psi4_step

This is the preferred method to install Psi4
plug-in, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for the Psi4 plug-in can be downloaded
from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/molssi-seamm/psi4_step

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/molssi-seamm/psi4_step/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/molssi-seamm/psi4_step
.. _tarball: https://github.com/molssi-seamm/psi4_step/tarball/master
